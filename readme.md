**TinkLibrary** 是**fengalngyj** 封装的一套常用工具类库，方便快速构建 **Web** 应用。

代码主仓库放在 **Gitee**，**Github** 仅为镜像仓库用于发布 **Composer** 包。

## 包含组件

* 【TelegramBotService】 Telegram 机器人相关服务

## 使用说明

* **TinkLibrary** 需要 **Composer** 支持
* 安装命令：`composer require fenglangyj/tink-library`
